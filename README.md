# JetClash Arena

Private planning repo for building JetClash Arena.

JetClash Arena is a fast 2D jetpack shooter with a phased approach:
1. **Phase 1 (Now):** fully playable MVP with in-game credits (no token dependency)
2. **Phase 2:** traction + retention validation
3. **Phase 3:** token launch on Bags with real in-game utility
4. **Phase 4:** progressively move core game authority on-chain using MagicBlock

# Phased Roadmap

## Phase 1: Playable MVP (no token)
- 1v1 mode + credits + leaderboard
- core retention loop validation

## Phase 2: Live Ops + polish
- balancing pass
- anti-abuse pass
- more maps/weapons
- PvAI survival mode

## Phase 3: Token launch on Bags
- token utility goes live (tickets, cosmetics, events)
- public treasury and transparency reporting

## Phase 4: MagicBlock migration
- move selected authoritative game state on-chain (starting with match settlement)
- progressively increase on-chain scope while preserving UX

## Phase 5: Full on-chain competitive layer
- ranked proof and verifiable match outcomes
- tournament rails and composable economy

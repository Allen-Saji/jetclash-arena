# MVP Scope (Phase 1)

## Must-have
1. Playable arena combat (1v1)
2. Match timer + score + win condition
3. Respawn system
4. 2 weapons + 1 ability
5. Health, ammo, pickups (health/ammo/speed)
6. Basic matchmaking queue (or private room code)
7. End-match result screen with rematch
8. In-game credits reward logic
9. Daily leaderboard (off-chain)

## Nice-to-have (if time)
- PvAI survival mode
- replay/highlight snippets
- profile stats page

## Explicitly out of scope for MVP
- token launch
- on-chain match authority
- complex clan system
- marketplace

## Definition of done
- stable gameplay with no hardlocks
- smooth 60fps target on average devices
- complete game loop from login to rematch
- meaningful reward/progression feedback

# Token Launch Plan (Bags) — Post-MVP

## Launch preconditions
Do not launch token until these are true:
1. playable MVP is live
2. retention baseline exists (D1/D7)
3. credits economy stable
4. basic anti-abuse controls live

## Token role
Token should have hard utility:
- tournament entry
- event tickets
- premium cosmetic unlocks
- seasonal pass

## Trust checklist for community
- public wallet list (team, treasury, rewards)
- vesting schedule for team allocation
- transparent use-of-funds plan
- weekly reporting cadence

## Funding policy
If low capital:
- small fair launch
- avoid overpromising
- prioritize shipping milestones over chart talk

## Post-launch discipline
- no random tokenomics changes
- publish weekly product + treasury updates
- tie incentives to real game activity

# Product Vision

## One-line pitch
JetClash Arena is a high-speed 2D jetpack combat game built for short, replayable sessions and competitive social ranking.

## Core principles
1. **Fun first**: gameplay quality before tokenomics.
2. **Skill first**: rewards should depend on performance, not grind volume alone.
3. **Fast loop**: one full loop (queue -> match -> result -> rematch) in under 3 minutes.
4. **Progressive decentralization**: start centralized enough to ship, then move authority on-chain via MagicBlock.

## Target players
- Web3-native gamers and degens
- short attention session players (mobile and desktop web)
- players who enjoy 1v1 competition and public leaderboards

## North-star outcome
A game people play because it is genuinely fun, where economy/token become multipliers, not crutches.

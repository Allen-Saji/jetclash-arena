# In-Game Credits Economy (Pre-Token)

## Currency
- `Credits` (off-chain, internal)
- no blockchain dependency in Phase 1

## Earn sources
- match completion
- win bonus
- streak bonus
- daily mission completion

## Suggested reward model
- Base complete reward: 20 credits
- Win bonus: +15
- Streak 3+: +10
- Daily mission: 30-80

## Spend sinks (MVP-safe)
- cosmetic unlocks
- profile flair
- ranked entry discount vouchers
- event tickets (later)

## Economy guardrails
- daily earn cap
- diminishing returns after many matches
- anti-abuse checks for repeated exploit patterns

## Goal
Prove that reward loops drive retention before introducing real token.

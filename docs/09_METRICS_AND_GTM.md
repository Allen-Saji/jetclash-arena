# Metrics and Go-to-Market

## Metrics to track from day 1
- DAU / WAU
- D1 retention
- D7 retention
- matches per user per day
- rematch rate
- average session length
- reward earn/spend ratio

## Product health thresholds
- D1 >= 30%
- D7 >= 12%
- rematch rate >= 40%

## GTM channels
1. X content (clips, scorecards, dev logs)
2. Discord playtest group
3. Referral invites from top players

## Content strategy
- post short gameplay clips daily
- share leaderboard snapshots
- highlight clutch moments and streaks

## Launch cadence
- Closed alpha (friends + skilled testers)
- Open beta with capped rewards
- Public launch once matchmaking + anti-abuse stable
